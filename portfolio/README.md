# Portfolio

Ce dossier contient votre portfolio basé sur `gatsby-theme-portfolio-minimal`.

## Installation

```bash
cd portfolio
yarn install
# ou
npm install
```

## Développement

```bash
yarn develop
# ou
npm run develop
```

Le site sera accessible sur `http://localhost:8000`

## Build

```bash
yarn build
# ou
npm run build
```

## Structure

-   `content/` - Contenu du site (articles, images, sections)
-   `src/pages/` - Pages personnalisées
-   `gatsby-config.js` - Configuration Gatsby
-   `package.json` - Dépendances

## Personnalisation

Modifiez les fichiers dans `content/` pour personnaliser votre portfolio :

-   `content/settings.json` - Métadonnées et configuration du site
-   `content/sections/` - Sections du portfolio (hero, about, projects, etc.)
-   `content/articles/` - Articles de blog
